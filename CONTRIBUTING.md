# Contributing to ADE

Thanks for your interest in contributing!

## Reporting Bugs

Open an [issue](https://github.com/alexsds/ade/issues) with:
- What you expected to happen
- What actually happened
- Steps to reproduce
- macOS version and hardware (Apple Silicon / Intel)

## Feature Requests

Open an [issue](https://github.com/alexsds/ade/issues) describing the use case and why it would be valuable.

## Pull Requests

1. Fork the repo and create a branch from `main`
2. Make your changes
3. Run `cargo fmt --all` and `cargo test`
4. Open a PR with a clear description of what changed and why

## Building

```bash
cargo build          # debug build
cargo test           # run tests
cargo fmt --all      # format code
```

Requires macOS and Rust toolchain (edition 2024).
